# Scope

## Purpose

Recruiter Agent helps a user improve application materials and run a disciplined job search while keeping personal career data private and traceable.

## In Scope

- Resume review, editing, formatting, and ATS-oriented readability.
- Job posting analysis from pasted or imported postings.
- Role expectation and skill-gap research.
- Company, compensation, and employee-review research with citations.
- Application workflow tracking and follow-up reminders.
- Local-first artifacts that can be inspected and exported.

## Out of Scope for Early Versions

- Automated job applications without explicit review.
- Scraping sites that prohibit automated access.
- Fabricating experience, credentials, dates, or accomplishments.
- Public product packaging before privacy and workflow maturity.

## Users

- People who want an assistant for truthful, research-backed job applications.
- Operators who need local-first testing before using private career data.
- Future teams packaging the workflow as an extension or standalone product.

## Acceptance Bar

Outputs should be useful, accurate, and reviewable. The assistant must distinguish source facts, user-provided facts, inferred fit, and uncertain recommendations.
